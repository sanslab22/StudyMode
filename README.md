# studymode

Study-mode web UI: login/sign-up flows, home with a Pomodoro-style timer, and a notes area with a folder/file sidebar. State lives in React context on the client (no API or database yet).

## Current status

- **Framework:** [Next.js](https://nextjs.org/) 15 (App Router) with **React 18**, **JavaScript** (no TypeScript in this repo).
- **Rendering:** Mostly client components (`'use client'`) with shared providers for timer and notes.
- **Routing:** File-based routes under `src/app` (`next/link`, `next/navigation`). Unknown URLs redirect to `/` via `src/app/not-found.js`.
- **Backend:** Not implemented—auth and data are UI-only placeholders.

## Scripts

## Folder structure

```text
studymode/
├── public/                 # Static assets (favicon, manifest, etc.)
├── src/
│   ├── app/                # Next.js App Router
│   │   ├── layout.js       # Root layout, metadata, global CSS
│   │   ├── globals.css     # Global styles (body, typography)
│   │   ├── providers.js  # Client wrapper: TimerProvider
│   │   ├── not-found.js    # Redirect unmatched routes to /
│   │   ├── page.js         # Route: / (Login)
│   │   ├── create-account/page.js
│   │   ├── forgot-password/page.js
│   │   └── (main)/         # Shared shell for logged-in-style views
│   │       ├── layout.js   # Layout + NotesProvider (sidebar + main)
│   │       ├── home/page.js
│   │       └── notes/page.js
│   ├── components/         # UI: Login, Home, Notes, Layout, Sidebar, …
│   └── context/            # TimerContext, NotesContext
├── next.config.mjs
├── package.json
└── .eslintrc.json
```

The `.next/` directory is generated by `next dev` / `next build` and is gitignored.

**Note:** Use **`src/app`** for the App Router only. Do not add an empty `app/` folder at the repository root—Next.js would prefer it over `src/app` and routes would 404.
